핵심 아이디어
1. 미리 작성한 딕셔너리에서 찾기 : '/'를 기준으로 2번 째 '/'와 3번 째 '/' 사이의 링크를 반영 ex) www.donga.com
2. meta[property="og:site_name"]로 찾기
3. meta[name="copyright"]로 찾기
4. meta[name="Copyright"]로 찾기

위의 우선순위대로 사이트를 찾는 로직입니다.
1~4번까지 방법으로도 찾지 못할 시에는 '검색불가'가 입력됩니다.

*코드작성이 완벽하지 않아, 엑셀 행 중 도중에 끊어져있는 행이 있을 경우에 그 부분에서 에러가 발생하는데, 이 부분의 수정이 필요할 거 같습니다.
